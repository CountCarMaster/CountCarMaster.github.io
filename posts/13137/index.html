对不同的模型拟合方法，我们使用二维的直线为例进行分析。

1 最小二乘法

对于二维的直线，其表达式为ax+by+c=0ax + by + c = 0ax+by+c=0

我们所期望的就是所给定的所有点到直线的距离之和最短，即损失函数为E=∑i=1n(axi+byi+c)2E = \sum_{i = 1}^n(ax_i + by_i + c) ^ 2\\E=∑i=1n​(axi​+byi​+c)2，具体如下图所示：

1.2 损失函数

最小二乘法会极大地受到噪声点的影响，只要有一个比较夸张的特征点就会把整个模型变得离谱。

因此，我们会选择其他的一些损失函数以降低外点（噪声点）对模型拟合的影响。可选的损失函数有：

  • L1. ρ=∣s∣\rho = |s|ρ=∣s∣
  • L2. ρ=s2\rho = s^2ρ=s2
  • Cauchy. ρ=log(1+∣s∣)\rho = log(1 + |s|)ρ=log(1+∣s∣)
  • Huber. ρ=s2(∣s∣<δ),2δ(∣s∣−12δ)(otherwise)\rho = s^2(|s| < \delta), 2\delta(|s| - \frac{1}{2}\delta)(otherwise)ρ=s2(∣s∣<δ),2δ(∣s∣−21​δ)(otherwise)
  • etc.

但使用之后，这就不再是一个线性问题了，因此也就不能用通用解法进行求解了。因此，我们需要使用以下方法：

  • 梯度下降法
  • 高斯-牛顿法
  • Levenberg-Marquardt

1.3 使用场景

这种方法又简单又方便，在没有外点/噪声点的情况下会很好用

但是当外点比较多或者里面不仅仅只有一个模型的时候，就不好用了。

2 霍夫变换

霍夫变换就是将坐标系变换为坐标轴为待求解的参数的坐标系。举例来说，就会是：

在左边的传统坐标系中，一条直线的表达式为y=ax+by = ax + by=ax+b，其中待求解的参数为aaa和bbb。我们进行简单变换，就会得到b=−xa+yb = -xa + yb=−xa+y，那么传统坐标系中的两个点就会变成右边坐标系中的两条直线，其中的交点就是准确的参数解。

但是我们不一定能够恰好找到交点，这个时候我们就会使用投票。举例来说，我们选定一个分辨率（精度），假如说为0.10.10.1，就会分出一个个大小为0.1×0.10.1 \times 0.10.1×0.1的格子。我们计算出a=0.1,0.2,...a = 0.1, 0.2, ...a=0.1,0.2,...所对应的bbb（精度条件下的下取整），相应格子得票数++，最终看看哪个格子里的票数最多，我们就采用这个参数。

但是对于直线而言，这种方法是存在问题的——垂直于xxx轴的直线无法表示。因此，我们可以使用极坐标，也就是直线使用xcos⁡θ+ysin⁡θ=rx\cos \theta + y\sin \theta = rxcosθ+ysinθ=r表示，这样我们需要求的参数就是θ\thetaθ和rrr。加入精度为0.10.10.1，那么θ\thetaθ就可以表示为0.1,0.2,...,359.9,3600.1, 0.2, ..., 359.9, 3600.1,0.2,...,359.9,360

那如果是需要三个参数的呢（比如说圆）？

圆的表示形式为(x−a)2+(y−b)2=r2(x - a) ^ 2 + (y - b) ^ 2 = r ^ 2(x−a)2+(y−b)2=r2

那么我们就使用下面的技巧：

3 RANSAC

这是一种效果很好，很简单，也能应对复杂模型的方法。

3.1 步骤

  • 随机选择能够表达该模型最少所需要的点。对于直线来说，需要使用两个点确定；对于平面而言则是三个点。
  • 用这些点解得这个模型。比如说我们用一般式来表达直线，即ax+by+c=0ax + by + c = 0ax+by+c=0，那么就可以通过这两个点得到a=y2−y1a = y_2 - y_1a=y2​−y1​，b=x1−x2b = x_1 - x_2b=x1​−x2​，c=x2y1−x1y2c = x_2y_1 - x_1y_2c=x2​y1​−x1​y2​
  • 计算每个点pi=(xi,yi)p_i = (x_i, y_i)pi​=(xi​,yi​)的error function（也就是到这个模型的距离）。直线的话也可以用法向量的方法di=nT(pi−p0)∣∣n∣∣2d_i = \frac{n^T(p_i - p_0)}{||n||_2}di​=∣∣n∣∣2​nT(pi​−p0​)​，其中法向量n=[−b,a]Tn = [-b, a] ^Tn=[−b,a]T
  • 数一数所有点里面距离大于我们的界限值得有多少，也就是di<τd_i < \taudi​<τ，这种叫外点，符合条件的则叫内点
  • 重复上面四步NNN次，选择其中内点最多的结果

3.2.2 NNN的确定

要确定这一参数，我们需要先知道下面内容：

  • eee: 外点率，也就是点是外点的概率
  • sss: 确定一个模型所需要的最少点数（如直线就是2）
  • NNN: 我们要求的RANSAC迭代次数
  • ppp: 我们至少能够得到一个好模型的置信度（0.99？0.999？）

那么(1−e)s(1 - e) ^ s(1−e)s就是选择sss个点都是内点的概率

1−(1−e)s1 - (1 - e) ^ s1−(1−e)s就是有一个或者多个点是外点的概率

(1−(1−e)s)N(1 - (1 - e) ^ s)^N(1−(1−e)s)N就是NNN次都不大行的概率，因此我们就有：

(1−(1−e)s)N=1−p(1 - (1 - e) ^ s)^N = 1 - p(1−(1−e)s)N=1−p，即N=log(1−p)log(1−(1−e)s)N = \frac{log(1 - p)}{log(1 - (1 - e)^s)}N=log(1−(1−e)s)log(1−p)​
